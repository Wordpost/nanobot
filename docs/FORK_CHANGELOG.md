# Fork Changelog

Этот файл содержит список уникальных изменений (fork-local), внесенных в репозиторий относительно оригинального (upstream).
Эти изменения необходимо сохранять при слиянии с апстримом. Измененные участки кода помечены комментарием `(fork-local)`.

## 2026-04-11
### `nanobot/agent/tools/web.py`
* **Fix**: Добавлена блокировка (`asyncio.Lock()`) в инструмент `web_search` для DuckDuckGo.
* **Причина**: При одновременном вызове метода поиска из нескольких потоков (например, когда модель решает сделать сразу два или более поисковых запроса) в Docker-окружении происходила взаимная блокировка (deadlock) внутри библиотеки `ddgs` (Rust-клиент `primp` зависал при инициализации соединений). Это приводило к тому, что весь сессионный цикл агента навсегда блокировался в ожидании `asyncio.to_thread`.
* **Решение**: Сериализация вызовов к `DDGS` с сохранением механизма `asyncio.wait_for` для защиты от зависших сетевых соединений DuckDuckGo.
